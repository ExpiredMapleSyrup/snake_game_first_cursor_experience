# 🐍 贪吃蛇游戏

一个使用HTML5 Canvas、CSS3和原生JavaScript开发的精美贪吃蛇网页游戏。

## 🎮 游戏特色

- **美观的配色方案**：鹅黄色(70%) + 浅粉色(20%) + 粉紫色(10%)
- **流畅的游戏体验**：60FPS流畅动画
- **智能按钮设计**：根据游戏状态自动切换
- **响应式设计**：适配不同屏幕尺寸
- **移动端支持**：支持触摸操作和虚拟控制按钮
- **本地存储**：最高分自动保存

## 🎯 游戏操作

### PC端操作
- **方向键** ↑↓←→ 控制蛇的移动
- **空格键** 暂停/继续游戏
- **开始游戏** 按钮开始新游戏
- **重新开始** 按钮重置游戏

### 移动端操作
- **虚拟方向按钮** 控制蛇的移动
- **虚拟暂停按钮** 暂停/继续游戏
- **触摸优化** 防止误操作和页面滚动

## 🎨 设计理念

游戏采用温馨的配色方案，营造舒适的游戏环境：
